# Movie Recommendation System

This project is a **simple movie recommendation system built using Python**.
It recommends similar movies based on user preferences using **cosine similarity**.

## Features

* Suggests movies similar to the selected movie
* Uses similarity analysis to find related movies
* Simple and beginner-friendly machine learning project

## Technologies Used

* Python
* Pandas
* Scikit-learn

## How It Works

1. Movie data is stored with features such as genre.
2. Cosine similarity is used to measure similarity between movies.
3. The system recommends movies that are most similar to the selected movie.

## How to Run the Project

Install required libraries:

```
pip install pandas scikit-learn
```

Run the program:

```
python movie_recommendation.py
```

## Example Output

If the input movie is:

```
Avengers
```

Recommended movies:

```
Iron Man
Thor
```

## Project Purpose

This project was built as a **beginner machine learning project** to understand recommendation systems and similarity algorithms.
